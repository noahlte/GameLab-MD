## C'est quoi le LUA ?

Le Lua n'est pas un langage de **programmation** à proprement parlé. En effet, c'est un langage de **scripting** qui est embarqué dans d'autres applications pour les **étendre** et leur ajouter des fonctionnalités.

### Pour mieux comprendre

Imagine que tu as un jouet comme des LEGO. Les briques LEGO de base, c'est le programme principal (comme un jeu vidéo). Le Lua, c'est comme des **briques supplémentaires** que tu peux ajouter pour personnaliser et améliorer ton jouet !

### Quelques exemples concrets

Le Lua est utilisé dans plein de domaines différents :

**Dans les jeux vidéos :**
- **Roblox** : Tous les jeux sur Roblox sont créés avec Lua !
- **Minecraft** (avec ComputerCraft) : Pour programmer des ordinateurs dans le jeu
- **World of Warcraft** : Pour créer des add-ons et des interfaces personnalisées
- **Angry Birds** : Le jeu a été développé en partie avec Lua
- **Garry's Mod** : Tous les mods sont écrits en Lua

**Dans d'autres applications :**
- **Extensions** de programmes en C++
- **Adobe Lightroom** : Pour automatiser des tâches de retouche photo
- **VLC Media Player** : Pour créer des extensions

### Pourquoi Lua a besoin d'une base ?

Le Lua a donc besoin d'une **base** pour fonctionner, c'est pour cela qu'on va installer des programmes supplémentaires. C'est un peu comme une application sur ton téléphone : elle a besoin du système d'exploitation (Android ou iOS) pour fonctionner.

### Les avantages du Lua

Le Lua est un langage très **simple** à comprendre et à apprendre, ce qui en fait un excellent choix pour débuter la programmation. Voici pourquoi il est si apprécié :

- **Facile à lire** : Le code ressemble presque à de l'anglais simple
- **Rapide** : Les programmes Lua s'exécutent très vite
- **Léger** : Lua prend très peu de place (parfait pour les jeux)
- **Flexible** : On peut créer plein de choses différentes
- **Très utilisé dans le jeu vidéo** : Beaucoup d'entreprises l'utilisent

### Le Lua, c'est pour qui ?

- **Débutants** : C'est un excellent premier langage de programmation
- **Créateurs de jeux** : Parfait pour créer des jeux rapidement
- **Moddeurs** : Idéal pour créer des mods de jeux existants
- **Professionnels** : Utilisé dans l'industrie du jeu vidéo

## Le LUA dans les jeux vidéos

Dans notre contexte, nous n'allons pas utiliser le Lua pour réaliser des plug-ins mais bien un **jeu entier** ! Pour cela nous allons utiliser des moteurs de jeu spécialisés.

### Moteurs de jeu utilisant Lua

**Love2D** 
- Moteur **gratuit** et **open-source**
- Parfait pour créer des jeux 2D
- Fonctionne sur **Windows, Mac, Linux, Android et iOS**
- Utilisé pour créer des vrais jeux commerciaux

**PICO-8** 
- Console de jeu **fantasy** (virtuelle)
- Limite volontairement les possibilités (comme les anciennes consoles)
- Parfait pour apprendre et créer des petits jeux rétro
- Possède une grande communauté qui partage des jeux

**Autres moteurs :**
- **Defold** : Pour créer des jeux mobiles et web
- **Corona SDK** : Spécialisé dans les jeux mobiles
- **Gideros** : Pour des jeux 2D et 3D
